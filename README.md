# ArchiveX 📦

ArchiveX is a premium physical file and asset management system built with React and Vite. It helps you keep track of where your physical documents, folders, and boxes are stored.

## Features
- **Smart Tracking**: Easily add and categorize physical items.
- **Advanced Search**: Search by name, location, or notes.
- **Dynamic Filtering**: Filter by item type or category.
- **Premium UI**: Modern design with fluid animations and responsive layout.
- **Persistence**: All data is stored locally in your browser.

## Tech Stack
- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 8](https://vitejs.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: Modern CSS with glassmorphism and custom design tokens.

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
